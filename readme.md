# admidori/dotfiles  
## Basic installation  
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/admidori/dotfiles/master/installer/installer.sh)"
```

## Manual installation  
```
git clone https://github.com/admidori/dotfiles.git
cd dotfiles
make install
```  

## Requirements software
- bash
- GNU make
- wget
- git
